# Анализ динамики показателей финансовой отчетности предприятий

Данный проект предназначен для анализа динамики финансовых показателей предприятий холдинга.

Система позволяет:
- хранить информацию о предприятиях;
- учитывать финансовые показатели;
- анализировать изменения показателей за отчетные периоды;
- оценивать эффективность деятельности предприятий.

В проекте используются показатели финансовой отчетности, единицы измерения и коэффициенты важности.
